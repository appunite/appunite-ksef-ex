Why KSeF Hub

Every input. One source of truth.

Most KSeF tools stop at download. KSeF Hub is the ledger that sits above the protocol: unified, categorized, permissioned, and API-addressable.

Live KSeF sync

XAdES-signed requests every 15 minutes. Certificate management, rotation, and revocation handled for you.

Foreign expenses, same ledger

Upload a PDF or forward an email. Hub parses, converts currency, and files it next to KSeF invoices.

ML categorization

87% of invoices auto-categorized with visible confidence scores. Adjust once; the model remembers.

Duplicate detection

Same NIP, same brutto, same month? We flag it before your accountant sees it twice.

Accountant access

Invite your księgowa with read-only or edit permissions. Per-invoice visibility controls built in.

API for everything

REST endpoints for invoices, expenses, and categories. Webhooks fire on sync. BI tools plug in directly.

The ledger

Unified row, whatever the source.

KSeF, PDF, email drop, API ingest, manual entry -- every invoice lands with the same shape. Same filters, same categories, same permissions. Your accountant stops asking which folder the Stripe invoice is in.

  • *Shared by default. One ledger per company; team members get scoped views.
  • *Fully auditable. Every change is logged. KSeF XAdES signatures preserved.
  • *Exportable. CSV, Excel, JSON, or live via API. No lock-in.
