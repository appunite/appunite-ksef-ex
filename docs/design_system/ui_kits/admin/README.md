# Admin UI kit

LiveView admin recreation. Based directly on `lib/ksef_hub_web/components/core_components.ex`, `layouts.ex`, `invoice_components.ex` and the LiveViews in `lib/ksef_hub_web/live/`.

Covers:
- App shell (top navbar, company selector, avatar menu, theme toggle)
- Invoices list with filters + pagination
- Invoice detail with details table, status banners, correction panel
- Sync history
- Certificates settings + expiry alerts
- Login / empty states

Run `index.html`. Click the nav to switch pages.
